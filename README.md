# 🌟 Korea MCT 온라인 벙개 2025년11월 🌟

## 📅 행사 개요

- **일시**: 2025년 11월 21일(금) 저녁 9시 (21:00–22:30)
- **장소**: Microsoft Teams
- **대상**: 전·현직 MCT 및 관심 있는 누구나
- **주관**: 박민진님

## 📝 프로그램 (Agenda)

1. 📸 **도전~! 다함께 사진! (첫번째)** — 진행
2. 🤝 **얼굴 익히기 (자기소개)** — 전체 참여
3. 🔄 **MCT 프로그램 업데이트** — 안성진님
4. 📖 **책 소개** — 김종덕님
5. 💡 **Ignite의 AI 중간 정리** — 윤금재님
6. 🔐 **Global Secure Access** — 우진환님
7. 🎯 **Team Activity** — 유승호님
8. 📸 **도전~! 다함께 사진! (두번째)** — 진행
9. 🗨️ **마무리 & 자유 네트워킹**

## 🎉 행사 진행 결과 (작성 예정)

- 총 **20명** 참석
- 커뮤니티 네트워킹
- MCT 프로그램 업데이트
- 기술 인사이트 (Ignite AI)
- 보안 & 접근 제어 (Global Secure Access)
- 책 소개 & 지식 공유
- 참여형 액티비티 진행 (Team Activity)
